To run the program

>>> flask db init
>>> flask db migrate -m "db init"
>>> flask db upgrade


>>> pipenv install
>>> pipenv shell
>>> cd server
to seed the database:
>>> python seed.py
to start the backend server:
>>> python app.py





To install and run app front end:

>>> npm install --prefix client
>>> npm run dev --prefix client
